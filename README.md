# committing

[![Greenkeeper badge](https://badges.greenkeeper.io/axetroy/committing.svg)](https://greenkeeper.io/)

每天定时commit，保持commit连击不间断...

当前是间隔一小时commit

### 使用

```bash
npm install committing -g
committing ./
# print the help information
committing --help
```
